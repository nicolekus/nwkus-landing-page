# nwkus-landing-page

## Overview
A simple static HTML landing page project.

## Project Structure
- `index.html` - Main HTML page (Hello World landing page)

## Running the Project
The project is served using Python's built-in HTTP server on port 5000.

**Workflow:** "Start application" runs `python3 -m http.server 5000`

## Deployment
Static site served via Python HTTP server.
